"This project is a wireless dam water level monitoring and flood early warning system.
An ultrasonic sensor measures the distance from the sensor to the water surface. 
The ESP32 calculates the water level and sends the information wirelessly using an nRF24L01 module.
At the receiver side, another ESP32 displays the water level and indicates the status as Safe Zone, Alert Zone, or Flood Danger using LEDs, an OLED display, and a buzzer.
